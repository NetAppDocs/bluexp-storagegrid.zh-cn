= 删除StorageGRID工作环境
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果您不想再从BlueXP管理StorageGRID 工作环境、请将其删除。

删除工作环境不会影响StorageGRID 系统。您可以随时从BlueXP中重新发现它。

.步骤
. 从导航菜单中、选择*存储>画布*。
. 在"画布"页面上、选择StorageGRID工作环境。
. 选择菜单图标，然后选择*从工作空间中删除*。
+
image:screenshot-remove.png["工作环境右上角菜单中提供的从工作空间删除选项的屏幕截图。"]

. 选择*Remove*确认。
